# Proyecto - Triple Espresso ☕ para Triple10

Sitio web para una cafetería ficticia, donde los usuarios pueden conocer la marca, reservar una mesa y encontrar información de contacto. Desarrollado como proyecto de práctica de HTML y CSS.


## 🛠️ Tecnologías usadas
- HTML5
- CSS3 (Flexbox, posicionamiento, animaciones)
- Git / GitHub

## 📋 Características
- Sección de inicio con presentación de la marca
- Formulario de reservación de mesa
- Sección de contacto
- Footer con enlaces a redes sociales
